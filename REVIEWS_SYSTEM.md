# نظام المراجعات - أنوار القرآن

## نظرة عامة

تم إضافة نظام مراجعات متكامل لتطبيق أنوار القرآن، يتيح للمستخدمين تقديم مراجعاتهم وآرائهم حول التطبيق. النظام مدمج مع Firebase للتخزين والإدارة.

## الميزات

### للمستخدمين
- نموذج مراجعة بسيط وسهل الاستخدام
- تقييم بالنجوم (1-5 نجوم)
- حقول للاسم والمهنة والنص
- رسائل تأكيد ورسائل خطأ واضحة
- دعم كامل للغة العربية (RTL)

### للمديرين
- لوحة إدارة مخصصة (`admin-reviews.html`)
- عرض المراجعات المعلقة والمعتمدة
- إمكانية اعتماد أو رفض المراجعات
- عرض تفاصيل كاملة لكل مراجعة
- واجهة سهلة الاستخدام

## الملفات المضافة

### الملفات الرئيسية
- `public/js/reviews.js` - منطق المراجعات مع Firebase
- `public/admin-reviews.html` - لوحة الإدارة
- `public/js/admin-reviews.js` - منطق لوحة الإدارة
- `firebase-rules.json` - قواعد أمان Firebase
- `deploy-firebase-rules.sh` - سكريبت نشر قواعد الأمان

### التحديثات
- `public/index.html` - إضافة قسم المراجعات
- `public/css/templatemo-style.css` - أنماط المراجعات

## الإعداد والتشغيل

### 1. نشر قواعد Firebase
```bash
chmod +x deploy-firebase-rules.sh
./deploy-firebase-rules.sh
```

### 2. اختبار النظام
1. افتح الموقع وانتقل إلى قسم "المراجعات"
2. املأ النموذج واختبر الإرسال
3. تحقق من Firebase Console للتأكد من حفظ البيانات

### 3. الوصول للوحة الإدارة
- انتقل إلى: `https://your-domain.com/admin-reviews.html`
- راجع المراجعات المعلقة
- اعتمد أو ارفض المراجعات حسب الحاجة

## قواعد الأمان

- **القراءة**: متاحة للجميع (لعرض المراجعات المعتمدة)
- **الكتابة**: متاحة للجميع (لتقديم المراجعات)
- **التحقق**: تأكد من صحة البيانات المدخلة
- **الحدود**: 
  - الاسم والمهنة: 100 حرف كحد أقصى
  - النص: 1000 حرف كحد أقصى
  - التقييم: 1-5 نجوم

## الأمان في الإنتاج

### حماية لوحة الإدارة
```javascript
// إضافة مصادقة للمديرين
function requireAdminAuth() {
    firebase.auth().onAuthStateChanged(function(user) {
        if (!user || !user.email.endsWith('@yourdomain.com')) {
            window.location.href = '/login.html';
        }
    });
}
```

### حماية من Spam
- إضافة reCAPTCHA
- تحديد عدد المراجعات لكل IP
- فلترة الكلمات المسيئة

## استكشاف الأخطاء

### مشاكل شائعة

1. **Firebase غير محمل**
   - تأكد من تحميل Firebase قبل reviews.js
   - تحقق من اتصال الإنترنت

2. **خطأ في الإرسال**
   - تحقق من قواعد Firebase
   - تأكد من صحة البيانات المدخلة

3. **لا تظهر المراجعات**
   - تحقق من حالة `approved`
   - تأكد من وجود مراجعات معتمدة

### سجلات التصحيح
```javascript
// تفعيل سجلات Firebase
firebase.database().ref('.info/connected').on('value', function(snap) {
    console.log('Firebase connected:', snap.val());
});
```
